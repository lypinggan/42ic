Markdown 语法

Markdown 语法及效果对比(右侧)

要输入斜体的话:

_以一个下划线开头的内容是斜体，可以包含空格_
*或者以一个星号开头*

要输入粗体的话:

__这里开始是粗体内容，可以有空格__
**或者以星号开头**

要输入标题的话:

# 这是一级标题
## 这是二级
...
###### 没错，这是六级标题

加入引用

以 > 开头的行全都是引用，例如
> 毛主席说过:
> 不以结婚位目的的恋爱都是耍流氓

列表

* Apple
* Microsoft
* Google

代码

要输入代码的话只需要每行前面有4个或四个以上的空格或者TAB，比如 

    int main(){
	    printf("Hello Markdown");
    }

在一行中加入代码的话只需要这样 `print "Python"`
(1 前面,TAB上面那个键)
